# myrtle.moe UI Kit — web app

Hi-fi recreation of the myrtle.moe web app shell. One product, this is it.

- `index.html` — interactive click-thru: home hero → operators grid → operator detail drawer.
- `Header.jsx`, `Footer.jsx`, `Hero.jsx`, `FeatureCard.jsx`, `OperatorGrid.jsx`, `Button.jsx`, `Badge.jsx`, `Card.jsx`, `Kicker.jsx`.

All components intentionally cosmetic — matches the original Tailwind v4 + base-ui styles as
closely as possible given the plain-CSS recreation. For production use, import the real
base-ui-react components from the repo.
