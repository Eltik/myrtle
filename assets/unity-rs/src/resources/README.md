## FMOD
Source: [FMOD Engine 2.02.22](https://fmod.com/download)

**Required Version:** FMOD Engine 2.02.22 (or any 2.02.xx version)

**For MinGW on Windows:**
- Download FMOD Engine from https://www.fmod.com/download
- Create MinGW import libraries using `create-fmod-libs.bat` (see `FMOD_INSTALLATION_GUIDE.md`)
- Copy `.dll` and `.a` files to `src/resources/FMOD/Windows/x64/`
